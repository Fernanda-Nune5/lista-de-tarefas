# 📝 Lista de Tarefas (To-Do List)

## 📌 Descrição
Aplicação web simples desenvolvida para gerenciar tarefas do dia a dia.  
Permite adicionar, marcar como concluídas e remover tarefas de forma prática, com salvamento automático no navegador.

---

## 🚀 Tecnologias utilizadas
- HTML  
- CSS  
- JavaScript  
- LocalStorage (armazenamento no navegador)

---

## 💡 Funcionalidades
- ✔ Adicionar novas tarefas  
- ✔ Marcar tarefas como concluídas (com efeito visual)  
- ✔ Remover tarefas da lista  
- ✔ Persistência de dados (as tarefas não somem ao recarregar a página)  
- ✔ Interface responsiva (funciona em celular)  
- ✔ Interações visuais (hover, focus e animações)

---

## 🎨 Interface
- Layout centralizado com Flexbox  
- Estilização com efeitos de hover e foco  
- Feedback visual ao concluir tarefas  
- Animações suaves com `transition` e `transform`  

---

## 📚 Conceitos aplicados
- Manipulação do DOM  
- Eventos (`click`)  
- Criação dinâmica de elementos (`createElement`)  
- Uso de classes com `classList`  
- Armazenamento com `localStorage`  
- Flexbox para layout  
- Pseudo-classes (`:hover`, `:focus`)  
- Pseudo-elementos (`::placeholder`)  
- Responsividade com `@media`

---

## ▶️ Como executar
1. Faça o download ou clone este repositório  
2. Abra o arquivo `index.html` em qualquer navegador
